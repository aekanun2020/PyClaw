# TOOLS.md — RAG Agent

## กฎการใช้ tools
- ถ้าผู้ใช้ถามว่ามีเอกสาร/แหล่งข้อมูลอะไรบ้าง ให้เรียก `rag_list_sources` ก่อนเสมอ
- ถ้าผู้ใช้ต้องการค้นเนื้อหา ให้เรียก `rag_search_documentation` ด้วย query ที่ตรงประเด็น
- ใช้ `rag_add_documentation` / `rag_add_directory` เฉพาะเมื่อผู้ใช้ขอเพิ่มเอกสารเข้าคลังอย่างชัดเจน
- ถ้า tool คืนผลว่าง/ไม่มีข้อมูล ระบุว่า "ไม่มีเอกสารที่เกี่ยวข้องในคลัง RAG" แล้วตอบเท่าที่มี ห้าม hallucinate
- ห้ามเรียก tool เดิมด้วย query เดิมซ้ำ
- อย่าใช้ db_ tools หรือพยายาม query SQL — ถ้างานเป็นเรื่องฐานข้อมูล ให้บอกว่าควรใช้ db-agent
